# DesignKit‑Project 🧩

A modern Design System Starter Template built with React, Vite, Tailwind CSS, and Shadcn UI. Provides reusable UI components, design tokens, and accessibility-ready elements for fast frontend development.

## Features
- Reusable UI components (Buttons, Inputs, Cards, Modals, Navbar, etc.)
- Design tokens for colors, typography, and spacing
- Accessibility-ready (keyboard navigation, ARIA attributes)
- Clean and responsive layout
- Fast development with Vite
- Easy customization for your project

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Shadcn UI
- **Design System:** Tokens, reusable components
- **Icons:** Lucide React
- *Optional:** TypeScript support
